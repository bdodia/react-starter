# react-starter

Basic react app

# Initialise project

- Clone the project
- Run <code>npm run install</code>

# live-server

Either

<code>yarn serve</code> or
<code>npm run serve</code>

# hot reload

Run the following command from the base directory to run hot build reload

Either

<code>yarn build</code> or
<code>npm run build</code>
